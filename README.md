## Слои

| # | Название | Активация | Содержимое |
|---|----------|-----------|------------|
| 0 | **Base** | По умолчанию | QWERTY, модификаторы |
| 1 | **Lower** | Удержание `LOWER` | F1–F12, символы, скобки |
| 2 | **Raise** | Удержание `RAISE` | BT профили, навигация, стрелки |
| 3 | **Adjust** | `LOWER` + `RAISE` | RGB, BT CLR, EP TOG |

> **Adjust** активируется одновременным удержанием `LOWER` (левая рука) и `RAISE` (правая рука).

## Энкодеры

| Энкодер | Действие | Слой |
|---------|----------|------|
| Левый ↑ | Громкость + | все |
| Левый ↓ | Громкость − | все |
| Левый нажатие | Mute | 0 |
| Правый ↑ | Page Up | все |
| Правый ↓ | Page Down | все |
| Правый нажатие | RGB Вкл/Выкл | 0 |

## RGB подсветка

Управление через **Layer 3 (Adjust)**:

| Клавиша | Действие |
|---------|----------|
| `RGB TOG` | Вкл / Выкл подсветку |
| `RGB BRI` | Яркость + |
| `RGB BRD` | Яркость − |
| `RGB EFF` | Следующий эффект |
| `RGB HUI` | Оттенок + |
| `RGB HUD` | Оттенок − |
| `RGB SAI` | Насыщенность + |
| `RGB SAD` | Насыщенность − |
| `EP TOG` | Питание RGB (полное отключение) |

### Эффекты (по порядку через EFF)

| # | Эффект |
|---|--------|
| 0 | Статичный цвет |
| 1 | Дыхание |
| 2 | Радуга |
| 3 | Волна ★ (стартовый) |
| 4 | Искры |

> Стартовый эффект задан в конфиге: `CONFIG_ZMK_RGB_UNDERGLOW_EFF_START=3`
> Максимальная яркость: `CONFIG_ZMK_RGB_UNDERGLOW_BRT_MAX=60`

## Bluetooth

Клавиатура поддерживает **5 независимых BT профилей**.

| Клавиша | Действие |
|---------|----------|
| `BT SEL 0` | Профиль 1 |
| `BT SEL 1` | Профиль 2 |
| `BT SEL 2` | Профиль 3 |
| `BT SEL 3` | Профиль 4 |
| `BT SEL 4` | Профиль 5 |
| `BT CLR` | Сбросить текущий профиль |

### Порядок включения

1. Включи **правую** половину
2. Подожди 3–5 секунд
3. Включи **левую** половину
4. BLE реклама начнётся автоматически

### Если клавиатура не подключается

1. Войди в Adjust (LOWER + RAISE)
2. Нажми `BT CLR`
3. Удали устройство на компьютере
4. Подключи заново

> Тестировать BLE нужно **без USB** — при подключённом USB ZMK отдаёт приоритет проводному подключению.

## Прошивка

### Сборка

Сборка происходит автоматически через GitHub Actions при пуше в `config/`.

```yaml
# build.yaml
include:
  - board: nice_nano@2.0.0//zmk
    shield: sofle_left
  - board: nice_nano@2.0.0//zmk
    shield: sofle_right
  - board: nice_nano@2.0.0//zmk
    shield: settings_reset
```

### Прошивка контроллера

1. Дважды нажми кнопку **Reset** — появится диск `NICENANO`
2. Скопируй `.uf2` файл на диск
3. Контроллер перезагрузится автоматически

| Файл | Назначение |
|------|------------|
| `sofle_left-...-zmk.uf2` | Левая половина |
| `sofle_right-...-zmk.uf2` | Правая половина |
| `settings_reset-...-zmk.uf2` | Полный сброс BLE |

### Сброс настроек BLE

При проблемах с подключением:

1. Прошей `settings_reset` на **левую** половину → подожди 10 сек
2. Прошей `settings_reset` на **правую** половину → подожди 10 сек
3. Прошей рабочую прошивку обратно на обе половины
