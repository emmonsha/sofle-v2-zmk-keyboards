## Слои

| # | Название | Активация | Содержимое |
|---|----------|-----------|------------|
| 0 | **Base** | По умолчанию | QWERTY, модификаторы, `_` |
| 1 | **Lower** | Удержание `LOWER` | F1–F12, символы, скобки |
| 2 | **Raise** | Удержание `RAISE` | BT профили, навигация, стрелки, emoji |
| 3 | **Adjust** | `LOWER` + `RAISE` | RGB, BT CLR, запись экрана, блокировка |

> **Adjust** активируется одновременным удержанием `LOWER` (левая рука) и `RAISE` (правая рука).

│ Shift │ Z │ X │ C │ V │ B │Ctrl+Space│  │RGB TOG │ N │ M │ , │ . │ / │Sft│

- Крайняя правая верхняя клавиша — `_` (нижнее подчёркивание)
- Левая кнопка энкодера — смена языка `Ctrl+Space`
- Правая кнопка энкодера — RGB вкл/выкл

- Правая кнопка энкодера — Screenshot `Cmd+Shift+4`

## Layer 3 — Adjust

> Активируется одновременным удержанием **LOWER + RAISE**.

```
┌────────┬──────┬──────┬──────┬──────┬──────┐                      ┌─────┬─────┬─────┬─────┬─────┬─────┐
│ BT CLR │ BT 1 │ BT 2 │ BT 3 │ BT 4 │ BT 5 │                      │ --- │ --- │ --- │ --- │ --- │ --- │
├────────┼──────┼──────┼──────┼──────┼──────┤                      ├─────┼─────┼─────┼─────┼─────┼─────┤
│EP TOG  │ HUD  │ HUI  │ SAD  │ SAI  │ EFF  │                      │ --- │ --- │ --- │ --- │ --- │ --- │
├────────┼──────┼──────┼──────┼──────┼──────┤                      ├─────┼─────┼─────┼─────┼─────┼─────┤
│  ---   │ BRD  │ BRI  │ ---  │ ---  │ ---  │                      │ --- │ --- │ --- │ --- │ --- │ --- │
├────────┼──────┼──────┼──────┼──────┼──────┼──────────┐  ┌────────┼─────┼─────┼─────┼─────┼─────┼─────┤
│  ---   │ ---  │ ---  │ ---  │ ---  │ ---  │Cmd+Sh+5  │  │Cmd+C+Q │ --- │ --- │ --- │ --- │ --- │ --- │
└────────┴──────┴──────┼──────┼──────┼──────┼──────────┤  ├────────┼─────┼─────┼─────┼─────┼─────┘
                       │ ---  │ ---  │ ---  │   ---    │  │  ---   │ --- │ --- │ --- │ --- │
                       └──────┴──────┴──────┴──────────┘  └────────┴─────┴─────┴─────┴─────┘
    [Энк L: громкость ± | кнопка: запись экрана]  [Энк R: вкладки браузера | кнопка: блокировка]
```

**Особенности:**
- `Cmd+Sh+5` — запись экрана `Cmd+Shift+5` (кнопка левого энкодера)
- `Cmd+C+Q` — блокировка экрана `Cmd+Ctrl+Q` (кнопка правого энкодера)

## Энкодеры

### Поворот

| Слой | Левый энкодер | Правый энкодер |
|------|--------------|----------------|
| 0 Base | Курсор ↑ / ↓ | Page Up / Page Down |
| 1 Lower | Undo `Cmd+Z` / Redo `Cmd+Shift+Z` | Курсор ← / → |
| 2 Raise | Масштаб `Cmd+=` / `Cmd+−` | Назад `Cmd+[` / Вперёд `Cmd+]` |
| 3 Adjust | Громкость + / − | Вкладки `Cmd+Shift+]` / `Cmd+Shift+[` |

### Нажатие кнопки

| Слой | Левая кнопка | Правая кнопка |
|------|-------------|---------------|
| 0 Base | Смена языка `Ctrl+Space` | RGB Вкл/Выкл |
| 1 Lower | Play / Pause | Screenshot `Cmd+Shift+4` |
| 2 Raise | Spotlight `Cmd+Space` | App Switcher `Cmd+Tab` |
| 3 Adjust | Запись экрана `Cmd+Shift+5` | Блокировка `Cmd+Ctrl+Q` |

## RGB подсветка

Управление через **Layer 3 (Adjust)**:

| Клавиша | Действие |
|---------|----------|
| `RGB TOG` | Вкл / Выкл (кнопка правого энкодера на L0) |
| `RGB BRI` | Яркость + |
| `RGB BRD` | Яркость − |
| `RGB EFF` | Следующий эффект |
| `RGB HUI` | Оттенок + |
| `RGB HUD` | Оттенок − |
| `RGB SAI` | Насыщенность + |
| `RGB SAD` | Насыщенность − |
| `EP TOG` | Питание RGB — полное отключение |

### Эффекты (переключение через `EFF`)

| # | Эффект |
|---|--------|
| 0 | Статичный цвет |
| 1 | Дыхание |
| 2 | Радуга |
| 3 | Волна ★ стартовый |
| 4 | Искры |

> Стартовый эффект: `CONFIG_ZMK_RGB_UNDERGLOW_EFF_START=3`
> Максимальная яркость: `CONFIG_ZMK_RGB_UNDERGLOW_BRT_MAX=60`

## Bluetooth

Поддерживается **5 независимых BT профилей**.

| Клавиша | Действие |
|---------|----------|
| `BT SEL 0` | Профиль 1 |
| `BT SEL 1` | Профиль 2 |
| `BT SEL 2` | Профиль 3 |
| `BT SEL 3` | Профиль 4 |
| `BT SEL 4` | Профиль 5 |
| `BT CLR` | Сбросить текущий профиль |

### Порядок включения

1. Включи **правую** половину
2. Подожди 3–5 секунд
3. Включи **левую** половину
4. BLE реклама начнётся автоматически

### Если клавиатура не подключается

1. Войди в Adjust (LOWER + RAISE одновременно)
2. Нажми `BT CLR`
3. Удали устройство на компьютере
4. Подключи заново

> Тестировать BLE нужно **без USB кабеля** — при подключённом USB ZMK переключается на проводной режим.

## Прошивка

### Сборка

Автоматически через GitHub Actions при пуше в `config/`.

```yaml
include:
  - board: nice_nano@2.0.0//zmk
    shield: sofle_left
  - board: nice_nano@2.0.0//zmk
    shield: sofle_right
  - board: nice_nano@2.0.0//zmk
    shield: settings_reset
```

### Прошивка контроллера

1. Дважды быстро нажми кнопку **Reset** — появится диск `NICENANO`
2. Скопируй `.uf2` файл на диск
3. Контроллер перезагрузится автоматически

| Файл | Назначение |
|------|------------|
| `sofle_left-...-zmk.uf2` | Левая половина |
| `sofle_right-...-zmk.uf2` | Правая половина |
| `settings_reset-...-zmk.uf2` | Полный сброс BLE памяти |

### Сброс BLE

При проблемах с подключением:

1. Прошей `settings_reset` на **левую** → подожди 10 сек
2. Прошей `settings_reset` на **правую** → подожди 10 сек
3. Прошей рабочую прошивку обратно на обе половины
