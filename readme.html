Слой Название Активация Содержимое
0 Base По умолчанию QWERTY, модификаторы, пробел, Enter
1 Lower Удержание LOWER F1–F12, символы, скобки, EP TOG
2 Raise Удержание RAISE BT профили, навигация, стрелки, Del
3 Adjust LOWER + RAISE RGB управление, BT CLR, EP TOG
